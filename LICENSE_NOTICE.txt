--- ENGLISH VERSION ---

This project reimplements and adapts bullet pattern ideas inspired by open-source bullet-hell engines.
While the JavaScript implementations are developed for this project, some bullet patterns and boss
movement logic may resemble those found in the referenced projects. 

References:
- Danmokou: https://github.com/Bagoum/danmokou (MIT)
- BulletUpHell: https://github.com/Dark-Peace/BulletUpHell (MIT)

This project acknowledges the artistic and gameplay influence of these projects,
including bullet pattern concepts, movement logic, and game mechanics.
It is **not intended to be an original creation**, but a reinterpretation in JavaScript.
